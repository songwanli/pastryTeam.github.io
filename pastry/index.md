# 关于

## 概述

Pastry平台是一套跨屏幕、跨平台的前端解决方案.支持主流的操作系统和浏览器,可以实现代码一次编写多平台运行.

## 特性

  * 跨平台
  * 易维护
  * 易安全
  * 高效率
  * 一站式集成开发
  * 易扩展

## 支持平台

* 操作系统支持
  * Windows XP 及以上
  * Linux
  * MacOSX
* 浏览器支持
  * IE 7+
  * Chrome
  * FireFox
  * Safari
* 移动平台支持
  * Android 2.3+
  * iOS 5.0+

## 开发模式

Pastry平台支持纯网页、微信及混合开发.

### 纯网页

仅使用网页功能,开发PC Web应用或移动端Wap应用.

### 微信

开发在微信中运行的页面.

### 混合

开发原生和Html混合开发的应用,Html页面可以通过桥调用原生提供的增强功能.

## 技术栈

Pastry平台基于以下开源框架

  * [Bootstrap][bootstrap]
  * [jQuery][jQuery]
  * [Backbone][Backbone]
  * [RequireJS][RequireJS]
  * [NodeJs][NodeJs]
  * [Gulp][Gulp]
  * [Cordova][Cordova]
