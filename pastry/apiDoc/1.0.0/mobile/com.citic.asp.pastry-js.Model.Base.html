Properties
Name Type Description
mode String 发送模式:
crossPage:是否允许请求跨页面
waitById:等待指定请求完成再发送当前请求
waitSame:如果有相同的请求,等待所有请求完成再发送当前请求
ignoreById:如果和指定请求相同,忽略当前请求
ignoreSame:如果有相同的请求,忽略当前请求
cancelById:如果和指定请求相同,取消该请求,并立刻发送当前请求
cancelSame:如果有相同的请求,取消所有相同请求,并立刻发送当前请求
id String 请求id,为0时代表所有请求
url String 数据请求的地址 server:开头为发送服务器请求
client:开头为发送客户端请求
initVal object 数据模型的初始化数据,每次数据刷新时都会先把数据初始化为该数据。
param object 数据请求的参数
wait boolean 是否为阻塞请求
autoFetch boolean 是否自动发送请求
timeout Number 数据请求的超时时间,为0时超时时间为无限长
mapping object 数据映射配置
