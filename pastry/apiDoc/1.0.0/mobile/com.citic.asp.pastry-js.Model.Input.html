Class: Model.Input

Model.Input

输入域数据模型

new Model.Input(defVal)

Parameters:
Name Type Description
defVal Object 数据默认值
Example
支持的校验属性:
     required    必填项
     minlen      最小数据长度
     maxlen      最大数据长度
     regex       指定数据格式(正则表达式)
     neq         指定数据不能和指定的相等
     eq          指定数据必须和指定的值相等
以上属性后增加Msg,可自定义错误提示,例如requiredMsg可指定必输项的错误提示
