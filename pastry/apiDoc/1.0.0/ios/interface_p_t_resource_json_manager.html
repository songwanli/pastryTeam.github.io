详细描述

功能：
1、更新 resource.json 文件，保证文件和服务器一致；
文件更新完毕，框架发送 [PTNotificationNameManager getNotificationName:PTNotification_RESOURCEJSON_UPDATE] 通知；
参数 PTEventState 枚举值；

[[NSNotificationCenter defaultCenter] addObserver:self
selector:@selector(receiveResourceJsonUpdateResult:)
name:[PTNotificationNameManager getNotificationName:PTNotification_RESOURCEJSON_UPDATE]
object:nil];


2、读取清单文件resource.json的内容
-—2.1 获取key-value键值对
-—2.2 获取version版本号

3、根据 resource.json 文件中 key value 键值对 对js，css，图片等文件，逐个验证这些文件的有效性
文件验证完毕，框架发送 [PTNotificationNameManager getNotificationName:PTNotification_FILELIST_CHECK_NAME] 通知；
参数 PTEventState 枚举值；

[[NSNotificationCenter defaultCenter] addObserver:self
selector:@selector(receiveCheckFileListResult:)
name:[PTNotificationNameManager getNotificationName:PTNotification_FILELIST_CHECK_NAME]
object:nil];
