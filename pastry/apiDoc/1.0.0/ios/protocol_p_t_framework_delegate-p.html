- (void) didHandShakeWithResult: (NSDictionary *)  dictionary
optional

密钥协商结果

参数
dictionary密钥协商结果 dictionary字典中可取的关键字有"result"、"data"、"serverError" 另外，当握手成功时还包含了所有服务器返回的握手信息
result : 0 表示密钥协商成功，此时data非空
1 表示密钥协商失败，此时serverError非空
2 表示密钥协商正在进行中
data : 握手成功 服务返回的数据
serverError:握手失败 错误参数；
