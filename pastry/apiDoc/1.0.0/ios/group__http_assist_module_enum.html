枚举类型说明

enum PTCommunicationErrorCode : NSUInteger

向服务器发送请求，返回请求状态枚举

CommunicationSuccess 连接成功
ParameterError 参数错误
错误的会话状态包括：
1、STATE_NULLSESSION 未建立会话
2、STATE_HANDSHAKEING 正在握手
3、STATE_HANDSHAKEFAILED 握手失败
4、STATE_SESSIONTIMEOUT 会话超时，需要重新握手
DataParseError 通讯结果加解密解析错误
NetworkUnreachableError 网络不通
SessionStateError 会话状态错误
ServerError 服务器错误
SessionTimeOut 会话过期
注解
只适用于 网络请求，服务器返回请求状态时；
不适用于 当前客户端所处的会话状态（客户端会话状态见 PTSessionState 枚举）；
参见
此枚举用于 PTCommunicationHelper 类
枚举值
CommunicationSuccess 

连接成功.

ParameterError 

参数错误.

NetworkUnreachableError 

网络不通.

SessionStateError 

会话状态错误.
错误的会话状态包括：
1、STATE_NULLSESSION
2、STATE_HANDSHAKEING
3、STATE_HANDSHAKEFAILED
4、STATE_SESSIONTIMEOUT

ServerError 

服务器错误.

DataParseError 

通讯结果解密解析错误.

enum PTSessionState : NSUInteger

会话状态标识枚举

STATE_NULLSESSION 未建立会话
STATE_HANDSHAKEING 正在握手
STATE_HANDSHAKEFAILED 握手失败
STATE_SESSIONTIMEOUT 会话超时，需要重新握手
STATE_UNLOGIN 已建立回话，握手成功，正在登陆
STATE_LOGING 沙箱目录下的 data.zip 文件路径
STATE_LOGIN 已建立会话，握手成功，登陆成功
STATE_LOGOUTING 正在登出
参见
此枚举用于 PTSessionManager 类
枚举值
STATE_NULLSESSION 

未建立会话

STATE_HANDSHAKEING 

正在握手

STATE_HANDSHAKEFAILED 

握手失败

STATE_SESSIONTIMEOUT 

会话超时，需要重新握手 : 服务器 状态

STATE_UNLOGIN 

已建立会话，握手成功，未登录 : 服务器 状态

STATE_LOGING 

已建立会话，握手成功，正在登陆 : 客户端 状态 （无）

STATE_LOGIN 

已建立会话，握手成功，登陆成功 : 客户端 状态

STATE_LOGOUTING 

正在登出 : 客户端 状态 （无）
