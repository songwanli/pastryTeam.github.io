# 平台插件

## pastry-plugin-browser 插件
* 功能
    * 提供多窗口功能，避免页面间的环境变量互相污染。
* 使用方法

## pastry-plugin-request 插件
* 功能
    * H5的网络请求通过原生加密后发送给服务器。
* 使用方法


## pastry-plugin-keyboard 插件
* 功能
    * 提供H5使用原生的密码键盘功能。
* 使用方法


## cordova-plugin-console 插件
* 功能
    * 提供H5使用 console.log() 将日志输出到 原生端的开发IDE输出框；
* 使用方法
